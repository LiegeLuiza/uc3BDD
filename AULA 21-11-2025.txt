3- Explique a diferença de um banco de dados, um sistema gerenciador de banco de dados e um sistema de banco de dados? 
Banco de dados: Coleção de dados logicamente coerente que possui significado. O sistema gerenciador gerencia o banco de dados.
 O sistema de banco de dados é um software que serve para criar, gerenciar armazenar e recuperar dados em um banco de dados

4- Quais são as principais características de um banco de dados? 
Padronização do acesso, Segurança do acesso, integridade das informações, escalabilidade, trabalho em equipe.
